# 📝 MCQ Test System

A web-based MCQ (Multiple Choice Questions) Test System that allows students to take quizzes and teachers to manage tests efficiently.

---

## 🚀 Features

### 👨‍🎓 Student Panel

* Attempt MCQ tests
* Instant result calculation
* User-friendly interface

### 👨‍🏫 Teacher Panel

* Create and manage quizzes
* Add questions dynamically
* Monitor student performance

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Firebase (for backend & database)

---

## 📂 Project Structure

```
├── index.html        # Landing page
├── student.html      # Student test interface
├── teacher.html      # Teacher dashboard
├── test.html         # Quiz page
├── css/              # Stylesheets
├── js/               # JavaScript logic
├── firebase-config.js
├── SPEC.md
```
